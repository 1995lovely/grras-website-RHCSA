Directory Configuration

Set up a collaborative directory with the following requirements:

  • Group ownership of /common/admin is to be set to sysadmin.
  • The directory should be readable, writable, and accessible only to members of sysadmin. No other user should have access.
  • Files created in /common/admin should automatically have group ownership set to the sysadmin group.

Note: It is understood that root has access to all files and directories on the system.
